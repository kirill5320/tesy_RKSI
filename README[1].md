# Сайт: Кулинарная книга "Рецепты бойца Сомы"

Веб-приложение для хранения и систематизации кулинарных рецептов.

## Команда
- ПМ: Пан Кира Витальевна
- Техлид / Бэкенд: Соломко Дмитрий Иванович
- Фронтенд: Ваня
- Тестировщик: Зенцов Кирилл Дмитриевич

## Технологии
- Бэкенд: Python + Django
- Фронтенд: HTML, CSS, JavaScript
- БД: PostgreSQL

## Функциональность
- Регистрация, авторизация
- CRUD рецептов
- Избранное
- Фильтры и категории
- Достижения
- Случайный рецепт

## Структура проекта
- /backend — серверная часть
- /frontend — клиентская часть
- /docs — документация
